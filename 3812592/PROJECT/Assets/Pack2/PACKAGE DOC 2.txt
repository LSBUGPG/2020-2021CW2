PACKAGE CONTENTS 2:

PICK UP & HOLD SYSTEM



DIRECTIONS OF SETUP:


CREATE EMPTY GAME OBJECT AS A CHILD OF THE PLAYER GAMEOBJECT
PUT IT A BIT FORWARD FROM THE PLAYER OBJECT
CREATE BULLET/BALL OBJECT
ADD A CUSTOM "Bullet" TAG
TAG BALL OBJECT AS BULLET
ADD A RIGIDBODY COMPONENT TO THE BALL
PUT PICK UP SYSTEM ONTO BALL OBJECT
SET "PICKUPSPOT" AS THE EMPTY OBJECT
SET "PICKUPSPOTOBJ" AS THE EMPTY OBJECT
SET POWER


DIRECTIONS OF USE:

CLICK & HOLD ONTO THE BALL TO HOLD IT
RELEASE MOUSE TO THROW IT
